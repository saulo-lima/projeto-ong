## 📌 About

This project is a front-end interface for an NGO registration system, focused on connecting organizations and making their data accessible in a simple and intuitive way.

The application allows users to:

* Register a new organization
* Access already registered partner entities
* Request new categories of activity when needed
* 
You can access the project here:
👉 https://saulo-lima.github.io/projeto-ong/

### 💡 Purpose

The main goal of this project is to create a clean and user-friendly interface that facilitates interaction between NGOs and potential partners, improving accessibility and organization of information.

### 🛠️ Technologies Used

* HTML5
* CSS3 (with Grid Layout and Flexbox)
* Google Fonts (Roboto)

### 🎨 Design & Structure

The layout was built using **CSS Grid**, separating the interface into two main areas:

* **Hero section** (visual + branding)
* **Form section** (actions and navigation)

The design focuses on:

* Clear visual hierarchy
* Highlighted actions using contrasting colors
* Responsive behavior for smaller screens

### 📱 Responsiveness

The project includes a responsive layout using media queries, adapting the interface for mobile devices by stacking sections and adjusting spacing.
